## Sales Performance Analysis

## Overview

This project analyzes retail sales data from the Sample Superstore dataset to uncover business insights using SQL, Python, and Microsoft Power BI.

The objective was to identify sales trends, evaluate profitability, analyze regional performance, and create an interactive dashboard for decision.

## Objective

- Analyze overall sales performance
- Identify the best-performance product categories
- Determine regional sales distribution
- Track monthly sales trends
- Visualize key business metrics
- Present findings through an interactive dashboard

## Tools Used

- Python
    - Pandas
    - Matplotlib
- Microsoft SQL Server(SSMS)
- Microsoft Power BI
- Microsoft Excel

## Project workflow

1. Cleaned and prepared the data using Python (Pandas).
2. Imported the dataset into SQL Server.
3. Performed SQL queries to answer business questions.
4. Create visualizations with matplotlib.
5. Built an interactive Power BI dashboard
6. Wrote a business report summarizing the findings.

## SQL Analysis

The SQL queries include:

- Total Sales
- Total Profit
- Top 10 Selling Products 
- Top 5 Customers 
- Sales by Categories
- Sales by Region
- Monthly Sales Trend
- Average Profit by Category

## Power BI Dashboard

The dashboard includes: 

- Total Sales KPI
- Total Profit KPI
- Number of Orders KPI
- Monthly Sales Trend Chart
- Sales by Category Chart
- Sales by Region Chart
- Region Slicer
- Category Slicer

## Key Insights

- Technology generated highest sales
- The West region contributed the largest share of sales
- Sales variewd throughtout the year
- Product categories showed different levels of probability
